직원 데이터 업로드

부서 등록 완료 후 직원 데이터를 업로드하세요.

  • 1아래 양식 다운로드를 눌러 직원 템플릿을 받아주세요.
  • 2B3 셀부터 데이터를 작성하고, 샘플 행(파란 배경)은 삭제 후 업로드하세요.
  • 3경혁팀장~평가제외 열은 0(해당없음) 또는 1(해당)만 입력하세요.
  • 4[Q열] 부서코드를 직접 입력하면 부서가 자동 등록됩니다 (부서 업로드 생략 가능).
  • 5업로드 전 미리보기에서 오류·경고를 확인한 뒤 확정하세요.
  • 6데이터 초기화는 복구 불가능합니다. 신중하게 사용하세요.
